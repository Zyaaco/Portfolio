# My Portfolio

This is my personal portfolio website, showcasing my projects and skills. The website is built using Next.js, TailwindCSS, and Framer Motion to create a fast, responsive, and visually appealing experience for visitors.

## Features

- **Next.js** for optimized server-side rendering and SEO
- **TailwindCSS** for customizable and responsive styling
- **Framer Motion** for smooth and engaging animations
